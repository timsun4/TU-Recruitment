# Recruitment
 Scheduling script for Greek Life Recruitment

## Outline
	
	

## Requirements
	1. Everyone is scheduled to attend parties as early as possible, that is, there is no waiting before an individual's first party or no waiting in between parties. 
	2. The number of people between each party for a chapter is as balanced as possible. 
	
	Note that requirement 1 takes priority over requirement 2. Additionally, priorty 2 will not hold strongly.

## Input
	A sample input has been provided in "PNM Schedule 1 Censored.xlsx". Assume the numbers represent people and the letters A-G represent chapter houses. The Group column in unchanged.

## Output
	The output is a spreadsheet with an indentical format the the input. Names should be alphabetical, and the schedule should reflect the requirements
